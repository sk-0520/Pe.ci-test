ヘルプ

インストール

特別なにか作業は必要ありません。 圧縮ファイルを展開すればインストールは完了します。

実行には .NET Framework(general)が必要です。

Pe 格納ディレクトリを移動する場合、スタートアップに登録している場合はディレクトリ移動前に登録を解除してください。

アンインストール

Peの配置されているディレクトリをそのまま削除すればアンインストールは完了します。

スタートアップに登録している場合はアンインストール前に登録を解除してください。

保存データ

通知領域(タスクトレイ)の Pe アイコンを右クリックし、 コンテキストメニューから 情報 / Pe 情報 を選択して「データ」ボタンを押下すると設定データが格納されたディレクトリがエクスプローラで開きます。
通常使用であれば設定ファイルやアップデート用アーカイブ等はすべてこのディレクトリに格納されるため完全に消してしまいたい場合はこのディレクトリを Pe 終了後に削除してください。

レジストリについて

以下のレジストリキーへ書き込みが行われます。

キー 使用目的
  • HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced
 隠しファイル・拡張子の表示切り替えに書き込みが行われます。
  • Hidden
  • HideFileExt
  • HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Main\FeatureControl\FEATURE_BROWSER_EMULATION
  • HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Main\FeatureControl\FEATURE_DOCUMENT_COMPATIBLE_MODE
 .NET Framework で使用するIEブラウザコンポーネントのユーザーエージェント、レンダリングモードを変更するために使用します。
  • PeMain.exe
  • PeMain.vshost.exe(デバッグ環境のみ)
Pe 起動時に書き込まれ、 Pe 終了時に削除されます。
